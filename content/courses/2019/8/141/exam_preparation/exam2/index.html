Exam 2 Study Guide

Material

  • Topics: Exam 2 will explicitly cover material from Ch.3

  • Questions: Exam 2 will consist of approximately 15-20 short-answer questions and 3-5 long-answer questions.

Learning Outcomes Being Tested

Below is the list of potential exam questions for each section covered in lecture.

Section 3.1

Additional Practice 3.1 #3-13(odd), 23, 29, 33, 35, 45, 51, 55 (textbook)

  1. Calculate the derivative of a constant function.
  2. Calculate the derivative of \(x^{n}\) using the power rule.
  3. Calculate the derivative of a function by first rewriting it as a power function (use negative exponents and eliminate root signs if necessary).
  4. Calculate derivatives of polynomials.
  5. Calculate derivatives of functions involving \(e^{x}\).
  6. Calculate second derivatives for any of the above functions.

Section 3.2

Additional Practice 3.2 #3-15(odd), 19, 23, 27, 29, 31, 49 (textbook)

  1. Differentiate a function using the Product Rule.
  2. Differentiate a function using the Quotient Rule.

Section 3.3

Additional Practice 3.3 #1-17(odd), 33, 35, 37 (textbook)

  1. State the derivative of all 6 trigonometric functions.
  2. Differentiate a function involving a trigonometric function.
  3. Use the product, quotient, and general power rules to show how the derivatives of \(\tan x\), \(\cot x\), \(\sec x\), and \(\csc x\) can all be found from the derivatives of \(\sin x\) and \(\cos x\).

Section 3.4

Additional Practice 3.4 #1-23(odd), 29, 51, 79, 97 (textbook)

  1. Compute the composition of two functions.
  2. Given a composite function, determine the inner function and the outer function.
  3. Calculate the derivative using the general power rule.
  4. Calculate the derivative using the general exponential rule.
  5. Calculate the derivative using the general trigonometric rule.
  6. Calculate derivatives using the chain rule more than once.
  7. Use the Chain Rule to differentiate exponential functions of the form \(b^x\).

Section 3.5

Additional Practice 3.5 #5, 7, 9, 23, 27, 31 (textbook)

  1. Use implicit differentiation to calculate \(\frac{dy}{dx}\) in terms of \(x\) and \(y\).
    • The differentiation may require use of the Product, Quotient, or Chain Rules.
  2. Use implicit differentiation to find the slope of the curve at a designated point.
  3. Determine the point where an implicitly defined function has a horizontal tangent.
  4. State the derivative of all 6 inverse-trigonometric functions.
  5. Differentiate a function involving an inverse-trigonometric function.
  6. Use implicit differentiation to calculate the derivative of an inverse-trigonometric function.

Section 3.6

Additional Practice 3.6 #3, 7, 9, 19, 33, 39, 41, 43 (textbook)

  1. Use implicit differentiation to differentiate the logarithmic function \(f(x)=\log_b x\).
  2. Calculate a derivative using the general logarithmic rule.
  3. Calculate derivatives of functions involving \(\ln(x)\).
  4. Simplify an expression using the logarithmic laws.
  5. Calculate the derivative of a function using the technique of logarithmic differentiation.

Section 3.8

Additional Practice 3.8 #3, 5, 9, 15 (textbook)

  1. Population Growth
    • Find the relative growth rate; find \(k\).
    • Determine the equation that describes the growth of a population.
    • Find the size of the population after a given amount of time.
    • Determine how long it will take for a growing population to reach a given size.
  2. Radioactive Decay
    • Find the equation that describe the radioactive decay of a sample.
    • Find the the relative decay rate given the half-life; find \(\lambda\).
    • Determine the remaining mass after a given number of years.
    • Determine how long it will take for a decaying object to reach a given mass.
  3. Newton's Law of Cooling
    • Find the equation that describe the temperature of a cooling object.
    • Determine the temperature after a given amount of time.
    • Determine how long it will take to reach a certain temperature.

Section 3.10

Additional Practice 3.10 #1, 5, 9, 11, 15, 27, 33 (textbook)

  1. Linear Approximations
    • Find the linearization of a function \(f(x)\) at a given \(x=a\).
    • Approximate a number using a linearization.
  2. Differentials
    • Compute the differential \(dy\) given a function \(y=f(x)\) (may require product, quotient, or chain rules)
    • Evaluate a differential \(dy\) given values for \(x\) and \(dx\).
    • Use differentials to approximate errors that occur due to approximate measurements (Compute the maximum possible error, relative error, and percentage error).
