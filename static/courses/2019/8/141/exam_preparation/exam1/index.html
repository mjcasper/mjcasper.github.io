Exam 1 Study Guide

Logistics

  • Date: Thursday, September 26, 2019

  • Time: 8:45pm-10:15pm ET - (it is a 1.5 hour exam)

  • Location: Your exam location is posted on UBlearns.

  • What to Bring: You should bring multiple pens / pencils, your approved calculator, and a photo ID.

    • Calculator: You are allowed to use only the Texas Instruments TI-30XIIS calculator on the exam. All other calculators are prohibited. See the Calculator and Reference Policy in the syllabus for more details.

    • Photo ID: You must bring a photo ID with you to your test site (either a UB student ID or a state-issued ID like a driver's licencse).

  • What Not to Bring: Do not bring any bags, books, notes, formula sheets, or other loose items.

    • No Formula Sheets: You are not allowed any outside references (e.g., cheat sheets, formula sheets, note cards, or textbooks). See the Calculator and Reference Policy in the syllabus for more details.

Material

  • Topics: Exam 1 will explicitly cover material from Ch.2

  • Questions: Exam 1 will consist of approximately 15-20 short-answer questions and 3-5 long-answer questions

Learning Outcomes Being Tested

Below is the list of potential exam questions for each section covered in lecture.

Section 2.1

Additional Practice Textbook problems: 2.1 #1, 3, 5, 7

Tangent Problem

  1. Find the slope of a secant line given points two \(P\) and \(Q\).
  2. Estimate the slope of a tangent line at a specified point on the curve:
    • Numerically using a table of secant slopes.
    • Graphically by drawing the tangent line.
  3. Find the equation of a tangent line at a specified point on the curve.

Velocity Problem

  1. Find the average velocity over a given time interval.
  2. Approximate the instantaneous velocity at a specified time given:
    • An equation \(y=s(t)\) that describes the position of the object.
    • A table of data for the position of the object over time.
    • A graph of the position function.

Section 2.2

Additional Practice Textbook problems: 2.2 #1, 5, 7, 9, 11, 19, 31, 41

  1. Numerical Methods:
    • Estimate the limit of a function given a table of values.
    • Fill in a table of function values to help Estimate the limit of a function.
  2. Graphical Methods:
    • Estimate the limit of a function given its graph.
  3. One-Sided Limits
    • Compute the limit of a piecewise function.
    • Determine if a limit exists by computing the one-sided limits.
  4. Infinite Limits
    • Find the infinite limit of a function given its graph.
    • Calculate the infinite limit(s) of a rational function.
    • Calculate the infinite limit(s) of a trigonometric function.
    • Calculate the infinite limit(s) of a logarithmic function.
    • Find the vertical asymptotes of a function.

Section 2.3

Additional Practice Textbook problems: 2.3 #3, 5, 9, 11, 13, 25, 31

  1. State the limit laws.
  2. Find the limit of a function using the limit laws.
  3. Find the limit of a function using the limit laws, and be able to justify each step by stating which limit law you used.
  4. Find the limit of a function using the additional techniques of:
    • factor and cancel
    • expand and cancel
    • multiplication by the conjugate
  5. Compute the limit of a piecewise function.
  6. Use the Squeeze Theorem to evaluate limits of certain functions.

Section 2.5

Additional Practice Textbook problems: 2.5 #3, 5, 11, 15, 17, 21, 35, 45, 53, 57

Continuity

  1. State the definition of continuous at a number \(a\).
  2. Determine the \(x\)-values where a function is continuous / discontinuous:
    • Graphically, given the graph of the function.
    • By checking the limit definition of continuity using limit laws.
  3. State the intervals where a function is continuous. For functions like:
    • polynomials, rational, root
    • trigonometric, inverse trigonometric
    • logarithmic, exponential
    • algebraic combinations of these functions (including composition)
  4. Use continuity to evaluate \(\displaystyle \lim_{x\to a} f(x)\) provided \(f\) is continuous at \(x=a\). (You must explain why the function is continuous by identifying the type of function and giving the intervals on which it is continuous.)

Intermediate Value Theorem

  1. State the Intermediate Value Theorem.
  2. Use the Intermediate Value Theorem to show that an equation has at least one solution inside a specified interval.

Section 2.6

Additional Practice Textbook problems: 2.6 #3, 5, 13, 15, 19, 23, 39, 47, 49

  1. Determine the limit at infinity of a function given its graph.
  2. Calculate limits at infinity (for functions like polynomials, rational, root, trig, \(e^{x}\), \(\ln x\), and \(\arctan x\))
  3. Compute the limit at infinity by using the \(1/{x^{n}}\) method and showing all work.
  4. Compute a limit by converting to a limit at infinity.
  5. Find the horizontal asymptote(s) of a function.

Section 2.7

Additional Practice Textbook problems: 2.7 #3, 5, 7, 9, 12, 13, 43, 47, 49

  1. Find the slope of the tangent line to the curve \(y=f(x)\) at a point \((a,f(a))\) using:
    • \(\displaystyle m=\lim_{x\to a} \tfrac{f(x)-f(a)}{x-a}\)
    • \(\displaystyle m=\lim_{h\to 0} \tfrac{f(a+h)-f(a)}{h}\)
  2. Find an equation of the tangent line to the curve \(y=f(x)\) at a given number \(a\).
  3. State the limit definition of \(f'(a)\).
  4. Use the limit definition of derivative to calculate \(f'(a)\).
  5. Given a position function for an object moving in a straight line, find the (instantaneous) velocity of the object at a specific time.
  6. Use the average rate of change to estimate the instantaneous rate of change of a function.

Section 2.8

Additional Practice Textbook problems: 2.8 #1, 3, 21, 23, 35, 37

  1. State the definition of the derivative of a function \(f\).
  2. Use the limit definition of derivative to calculate the derivative of:
    • A polynomial of degree 3 or less (multiply and cancel).
    • A root function (conjugate method).
    • A rational function (common denominator).
  3. Given the graph of the function \(f\):
    • Sketch a graph of the derivative function \(f'\).
    • Determine the numbers where the function is non-differentiable.
  4. Given information about the function \(f\) (either a graph of \(f\) or a table of values) estimate values of the derivative function \(f'\).
  5. Find the velocity or acceleration functions given the position function.
